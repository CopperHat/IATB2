El programa se ejecuto usando el IDE Pycharm.
Utiliza un entorno de Python 3.8, debido a las compatibilidades con las librerias que se utilizaran.
Ademas, dentro del programa eseta la carpeta venv, que es el virtual enviroment donde se pudo ejecutar con normalidad el programa.
Para su optima ejecucion, utilizar dicho entorno para probar e programa
